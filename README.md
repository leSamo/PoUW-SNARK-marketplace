# SNARK marketplace blockchain

## Requirements
- Python 3.11 with pip
- Zokrates 0.8.8

## First time setup
1. Make sure you run `pip install -r requirements.txt` to install dependencies
2. (Optional) Configure network parameters in `config.py`
